The Sequin brand mark: the **ST** monogram ring (echoing the embossed marble vessels), the **SEQUIN** wordmark, or the full lockup. Gold foil renders on any background; the wordmark inherits text color via tokens (so it flips automatically under `data-theme="noir"`).

```jsx
<Monogram variant="lockup" size={44} tagline />   // site header
<Monogram variant="mark" size={32} />              // favicon / compact
<Monogram variant="wordmark" size={64} />          // hero / footer
```

Use the lockup in headers, the mark where space is tight, the wordmark for large editorial signatures.
