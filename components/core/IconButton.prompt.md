Circular, icon-only button for utility actions (search, cart, favourite, carousel nav). Pass an inline SVG or Lucide icon as children and an accessible `label`.

```jsx
<IconButton label="Search"><i data-lucide="search" /></IconButton>
<IconButton variant="gold" label="Add to cart"><i data-lucide="shopping-bag" /></IconButton>
<IconButton variant="bare" size="sm" label="Favourite"><i data-lucide="heart" /></IconButton>
```

Variants: `default` (linen surface), `gold` (champagne foil), `bare` (transparent). Sizes `sm | md | lg`.
