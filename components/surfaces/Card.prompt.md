Versatile surface container with the brand's warm, layered elevation and a top inner highlight (looks lit from above).

```jsx
<Card>…</Card>
<Card interactive>…</Card>                 // lifts + gold hairline on hover
<Card variant="glass">…</Card>             // frosted, for overlays on photography
<Card variant="feature">…</Card>           // bordeaux gradient, light text
```

Variants: `default`, `glass`, `feature`. `padded` (default true), `interactive` for hover lift. Compose other primitives inside.
